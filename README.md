## 🛠 Stack Technique

- **Frontend :** React 18, Vite, TypeScript
- **Styling :** Tailwind CSS
- **Animations :** Framer Motion
- **Backend :** Supabase (Auth, Database, Storage)
- **i18n :** i18next (Français, Anglais, Allemand)
- **Images :** Cloudinary API
- **Notifications :** Web Push / VAPID

## ✨ Fonctionnalités

- 🌐 **Multilingue :** Support complet FR / EN / DE avec détection automatique.
- 📱 **Responsive Design :** Optimisé pour tous les écrans (Mobile, Tablette, Desktop).
- 🔒 **Dashboard Admin :** Interface sécurisée pour gérer les projets, les avis et les statistiques.
- 🖼 **Gestion de Médias :** Upload et optimisation d'images via Cloudinary.
- 🔔 **Notifications Push :** Système d'alerte en temps réel pour les nouveaux messages.
- 📄 **Projets confidentiels :** Option pour masquer certains projets avec un flou interactif.

## 🚀 Installation

1. **Cloner le repository :**
   ```bash
   git clone https://github.com/votre-repo/portfolio.git
   cd portfolio
   ```

2. **Installer les dépendances :**
   ```bash
   npm install
   ```

3. **Configurer les variables d'environnement :**
   Créez un fichier `.env` à la racine :
   ```env
   VITE_SUPABASE_URL=votre_url
   VITE_SUPABASE_ANON_KEY=votre_cle_anon
   VITE_CLOUDINARY_CLOUD_NAME=votre_cloud_name
   VITE_CLOUDINARY_UPLOAD_PRESET=votre_preset
   VITE_VAPID_PUBLIC_KEY=votre_cle_vapid
   ```

4. **Lancer le serveur de développement :**
   ```bash
   npm run dev
   ```

5. **Build pour la production :**
   ```bash
   npm run build
   ```
